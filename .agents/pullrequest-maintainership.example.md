# Pullrequest Maintainership Example

Copy this file to `.agents/pullrequest-maintainership.md` or `.github/pullrequest-maintainership.md`, then replace the example targets with your GitHub owner or repository.

Do not keep example rows in an active maintainership file.

| target | role | notes |
|---|---|---|
| owner/repo | maintainer | auto-merge allowed for medium mode |
| owner | not-maintainer | ask before merge |
